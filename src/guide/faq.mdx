### Make a rounded window
Rounded windows are simply transparent square ones with a rounded rectangle
inside of them.

```qml
@@Quickshell.PanelWindow {
  color: "transparent"

  @@QtQuick.Rectangle {
    // match the size of the window
    anchors.fill: parent

    radius: 5
    color: "white" // your actual color
  }
}
```
